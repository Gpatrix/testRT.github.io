Le régiment Post-Scriptum

Le régiment Post-Scriptum existe depuis 2021.
C'est un jeu FPS de simulation durant la Seconde Guerre Mondiale.
Les principales activités du régiments sont:
- Des games entres [RT] / Des entraînements / Des matchs / Evènements / ...
De plus la [RT] est championne Européenne en blindée sur Post-Scriptum
Le gérant de ce régiment est Remzer

Le régiment Squad

Les principales activités du régiments sont:
- Des games entres [RT] / Des entraînements / Evènements / Des matchs / ...

Le régiment Arma

Le régiment Arma existe depuis janvier 2023.
C'est un jeu FPS de simulation de Guerre Moderne Tactique, attention le gameplay n'a rien à voir avec Squad.
Les 2 jeux sont éxtrêmements différents.
Les principales activités du régiments sont:
- Des opérations ( comme une campagne de jeu vidéo) créée par le gérant du régiment.
Le gérant de ce régiment est Maitrecrabe, le co-gérant est renai28

Le groupe multigaming

Le groupe multigaming ou groupe gaming existe depuis aout 2023.
Ce groupe organise plusieurs fois par mois des soirées sur le serveur discord.
Ces soirées se déroulent soient sur des jeux online ou directement sur le serveur discord [RT].
Les principales activités du groupe sont:
- Evènements / soirées gaming / ...
Le gérant de ce régiment est papy1789, le co-gérant est ewaiz

Fonctionnement de la [RT]

Elle possède un chef qui représente l'équipe: RushATom, il dirige l'équipe dans son ensemble de plus, un groupe d'administration est là pour l'aider, il est en charge de l'aspect économique et administratif de l'équipe, Riley est le gérant de ce groupe administratif.

En ce qui concerne l'aspect gameplay, la [RT] est divisée en "régiments" tous gérés par un gérant et un co-gérant en charge des évènements et animations de celui ci, la [RT] est présente sur War-Thunder, Post-Scriptum, Squad, Arma III,...

"La bonne camaraderie mène à la Victoire"

Comment rejoindre la [RT]

Dans un premier temps il faut rejoindre le serveur discord.
Une fois dessus il vous faudra suivre les étapes qui vous seront proposés automatiquement sur le serveur.
En suivant ces étapes vous allez avoir un "ticket".
Un fois dans ce "ticket" il vous faudra attendre 48H avant de pouvoir passer un "entretien"
permettant d'intégrer la [RT].
